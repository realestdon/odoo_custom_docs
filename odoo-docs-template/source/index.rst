About This Documentation
------------------------

This documentation is organized into three main sections:

**User Guide**
   End-user documentation for using the custom modules. Start here if you're
   looking to learn how to use the features.

**Modules**
   Detailed documentation for each custom module including installation,
   configuration, and usage instructions.

**Developer Guide**
   Technical documentation for developers who need to maintain, extend, or
   integrate with these modules.
