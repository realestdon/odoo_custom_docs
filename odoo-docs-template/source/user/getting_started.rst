Getting Started
===============

Welcome to the custom Odoo modules! This guide will help you get started.

First Login
-----------

Accessing the System
~~~~~~~~~~~~~~~~~~~~

1. Open your web browser
2. Navigate to your Odoo instance URL
3. Enter your username and password
4. Click **Log in**

.. image:: /_static/images/login_screen.png
   :alt: Login Screen
   :align: center
   :width: 60%

Changing Your Password
~~~~~~~~~~~~~~~~~~~~~~

On first login, you should change your password:

1. Click on your name (top right)
2. Select **My Profile**
3. Click **Change Password**
4. Enter your new password twice
5. Click **Change Password**

Common Tasks
------------

Creating Records
~~~~~~~~~~~~~~~~

Most modules follow this pattern:

1. Click the **Create** button
2. Fill in required fields (marked with *)
3. Optionally fill other fields
4. Click **Save**

Searching for Records
~~~~~~~~~~~~~~~~~~~~~

1. Use the search bar at the top
2. Type your search terms
3. Press Enter or click the search icon
4. Results appear below

Filtering Records
~~~~~~~~~~~~~~~~~

1. Click **Filters** dropdown
2. Select pre-defined filters or
3. Create custom filters

Editing Records
~~~~~~~~~~~~~~~

1. Click on the record you want to edit
2. Click **Edit** button
3. Modify fields as needed
4. Click **Save**

Best Practices
--------------

Daily Usage
~~~~~~~~~~~

✅ **Do:**

* Save your work regularly
* Use meaningful names for records
* Add descriptions to help others
* Archive old records instead of deleting
* Use tags for organization

❌ **Don't:**

* Leave records in draft unnecessarily
* Delete records unless absolutely needed
* Share your password
* Keep multiple tabs open (can cause conflicts)

Data Quality
~~~~~~~~~~~~

* Enter complete information
* Double-check important fields
* Use consistent formatting
* Update records when information changes

Troubleshooting
---------------

Common Issues
~~~~~~~~~~~~~

**Can't Save Record**

* Check all required fields are filled
* Look for error messages in red
* Ensure you have permission to save

**Record Disappeared**

* Check your filters
* Look in archived records
* Search by name or reference

**System Slow**

* Close unused browser tabs
* Clear browser cache
* Contact IT if problem persists
