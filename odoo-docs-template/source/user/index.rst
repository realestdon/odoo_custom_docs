About
-----

This section contains end-user documentation for all custom Odoo modules.
Whether you're new to the system or looking for specific instructions,
you'll find the information you need here.

Who is this for?
----------------

* **End Users**: Daily users of the system
* **Team Leaders**: Managing teams using the modules
* **Business Users**: Making business decisions based on data

What you'll learn
-----------------

* How to navigate the interface
* How to perform common tasks
* Best practices for daily operations
* Tips and tricks for efficiency

Getting Help
------------

If you can't find what you're looking for:

1. Check the module-specific documentation in :doc:`/modules/index`
2. Review the :doc:`getting_started` guide
3. Contact your system administrator
4. Email support at support@yourcompany.com
